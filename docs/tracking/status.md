# 현황

## 완료

| 기능 | 상태 |
|------|------|
| 이메일 OTP 인증 (발송 · 검증 · JWT 발급) | 구현됨, 테스트 없음 |
| 책 CRUD (목록 · 상세 · 추가 · 수정 · 삭제) | 구현됨, 테스트 없음 |
| 문장 CRUD | 구현됨, 테스트 없음 |
| 컬렉션 CRUD + 책-컬렉션 연결 | 구현됨, 테스트 없음 |
| 컬렉션 완독 제약 (완독 책만 추가 가능) | 구현됨 (서버 400 + 클라이언트 버튼 숨김 + 오류 메시지) |
| 통계 (상태별 · 연도별 · 월별 · 분야별 · 평균 별점 · 총 문장 수) | 구현됨, 테스트 없음 |
| 라이트 / 다크 테마 (localStorage 유지) | 구현됨 |
| 화면 캡처 기능 (html2canvas) | 구현됨 |
| Vercel(프론트) + Railway(백엔드+DB) 배포 구성 | 완료 |
| PostgreSQL 전환 (SQLite에서 마이그레이션) | 완료 |
| CORS 도메인 제한 (ALLOWED_ORIGINS 환경변수) | 구현됨 |
| 인증 코드 발송 횟수 제한 (send: 분당 5회) | 구현됨 |
| 인증 코드 입력 횟수 제한 (verify: 분당 10회) | 구현됨 |
| 불필요한 import.js 스크립트 삭제 | 완료 |
